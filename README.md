# La Lupe - Web App

Proyecto web dinámico para la gestión y visualización de un restaurante mexicano, desarrollado con PHP, MySQL y SCSS.

## Características
* **Menú Digital:** Visualización de platos y categorías.
* **Sistema de Usuarios:** Registro e inicio de sesión funcional.
* **Diseño Responsive:** Adaptado a escritorio.
* **Panel Legal:** Incluye Políticas de Privacidad y Aviso Legal.

## 🛠️ Tecnologías utilizadas
* **Frontend:** HTML5, SCSS (Sass), JavaScript.
* **Backend:** PHP 8.x.
* **Base de Datos:** MySQL / MariaDB.
* **Servidor:** Apache (XAMPP en local / InfinityFree en producción).
